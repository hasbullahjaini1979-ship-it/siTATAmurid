⚙️ Konfigurasi Google Sheets

Hubungkan aplikasi ke Google Sheets sebagai database. Gunakan Google Apps Script sebagai backend API.

📋 Panduan Setup Google Sheets:
1. Buka Google Sheets → Ekstensi → Apps Script
2. Copy-paste kode Apps Script yang tersedia
3. Deploy sebagai Web App (Execute as: Me, Access: Anyone)
4. Copy URL yang diberikan dan paste di atas
5. Klik "Test Koneksi"
